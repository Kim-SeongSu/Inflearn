## 
> [!TIP]
> 🔆 **실무에 도움이 될 꿀팁 정리!**


- `단축키 모음`
> IntelliJ의 단축키
```java
01. 빠른 코드 실행: ctrl + shift + F10
02. 줄 전체 복사: ctrl + D
03. 클래스명 수정: workspace에서 이름 변경 후 Alt + Enter  → "Rename File"
04. 메서드 위치로 이동: 메서드 이름 클릭 + ctrl + B
05. 한번에 변수 변경: shift + F6
06. 반환 메서드 이름 자동 생성: Ctrl + Alt + V           (new)
```
